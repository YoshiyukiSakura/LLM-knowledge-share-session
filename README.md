## English

### Overview

An interactive presentation that demystifies Large Language Models (LLMs) by building understanding from first principles. The presentation takes you on a journey from simple mathematical rules to the cutting-edge Transformer architecture and AI agents.

### Presentation Structure

| Part | Topic | Key Concepts |
|------|-------|--------------|
| 01 | Understanding the World Through Rules | Input/Output correspondences, y = x |
| 02 | Rule Transformations & Software Boundaries | y = kx + b, Symbolicism, Connectionism |
| 03 | Breaking Through Linear Boundaries | Non-linear data, Activation functions |
| 04 | From Neurons to Transformer | Neural networks, Attention mechanism, Transformer architecture |
| 05 | Large Language Models | Text completion, Training & inference, Capabilities & limits |
| 06 | Agents & Tool Calling | Agent architecture, Function Call, MCP Protocol |
| 07 | Summary | From rules to intelligence |

### Key Highlights

- **Visual Learning**: Animated graphs and diagrams to illustrate complex concepts
- **Intuitive Analogies**: Real-world examples (eating fries → ketchup, cold weather → jacket)
- **Progressive Complexity**: Build from `y = x` all the way to Transformer architecture
- **Practical Applications**: Covers Agent development, Function Calling, and MCP Protocol

### Quick Start

```bash
# Clone the repository
git clone https://github.com/YoshiyukiSakura/LLM-knowledge-share-session.git
cd LLM-knowledge-share-session

# Run English version
cd presentation-en
npm install
npm run dev

# Or run Chinese version
cd presentation
npm install
npm run dev
```

### Navigation

- **Arrow Keys** (← →): Navigate between slides
- **Number Keys** (1-9): Jump to specific slide
- **Click Navigation**: Use on-screen controls

### Project Structure

```
LLM-knowledge-share-session/
├── presentation/        # Chinese version (中文版)
├── presentation-en/     # English version
├── vercel.json         # Deployment config
└── README.md
```

## 中文

### 概述

这是一个交互式演示文稿，通过从基本原理构建理解来揭开大语言模型（LLM）的神秘面纱。演示将带您从简单的数学规则到最前沿的 Transformer 架构和 AI 智能体。

### 演示结构

| 部分 | 主题 | 核心概念 |
|------|------|----------|
| 01 | 用规则理解世界 | 输入/输出对应关系，y = x |
| 02 | 规则的变换与软件的边界 | y = kx + b，符号主义，连接主义 |
| 03 | 突破线性的边界 | 非线性数据，激活函数 |
| 04 | 从神经元到 Transformer | 神经网络，注意力机制，Transformer 架构 |
| 05 | 大语言模型 | 文本补全，训练与推理，能力与局限 |
| 06 | Agent 与工具调用 | Agent 架构，Function Call，MCP 协议 |
| 07 | 总结 | 从规则到智能 |

### 核心亮点

- **可视化学习**：动画图表帮助理解复杂概念
- **直观类比**：生活化的例子（吃薯条 → 蘸番茄酱，天冷 → 穿外套）
- **循序渐进**：从 `y = x` 一路构建到 Transformer 架构
- **实战应用**：涵盖 Agent 开发、Function Calling 和 MCP 协议

### 快速开始

```bash
# 克隆仓库
```
